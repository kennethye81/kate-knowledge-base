台大医院在宅医疗中心 — 27 维度深度尽调报告

调研日期：2026-07-11
实体类型：Track A — 公立医学中心自有 HaH 项目（非独立公司）
市场环境：台湾单一支付方（全民健保 NHI）体系
数据截止：2026 年 7 月（基于公开资料 + 政策文件）
语言：简体中文

D0 — 赛道真实性验证

核心判断：✅ 确认属于 Hospital at Home (HaH) 赛道

台大医院在宅医疗中心是台湾最早系统化开展「居家急性医疗」（Hospital at Home / 在宅住院）的三甲医院项目。其服务覆盖急性感染居家抗生素治疗（相当于美国 CMS Acute Hospital Care at Home 模式）、居家安宁疗护、慢性伤口照护等，属于 True HaH（急性住院替代），而非单纯的慢性 RPM 或传统居家护理。

赛道定位： | 维度 | 判断 | |:------|:-----| | 是否为急性住院替代？ | ✅ 是——提供肺炎/尿路感染/软组织感染的居家 IV 抗生素治疗 | | 是否包含 24h 响应能力？ | ✅ 是——照护小组提供 24 小时电话咨询 + 紧急访视 | | 是否由医院直接运营？ | ✅ 是——台大医院（公立医学中心）直接管理 | | 对标美国 HaH 公司？ | ⚠️ 否——公立系统内部项目，非商业实体，但模式可对标 DispatchHealth/Contessa 的临床方案 |

赛道误分类警示：部分台湾产业报道将传统「居家护理」（skilled nursing）与「在宅急症照护」混为一谈。本篇仅聚焦台大医院的急性 HaH 部分。

D1 — 执行摘要

一句话定位：台大医院在宅医疗中心是台湾 Hospital at Home 模式的先驱——2016 年起以三甲医院资源系统化开展急性居家医疗，2024 年 NHI 全国试办计划即以台大经验为核心蓝本。

核心矛盾：

  • 短期逆风：台湾护理人力严重短缺，全台医院被迫关床，HaH 模式虽可释放床位但需要额外医护人力投入，短期内难以大规模扩张
  • 长期逻辑：2025 年台湾已进入超高龄社会（65 岁以上人口 >20%），97.7 万独居老人中 5 万人需照护；医院床位有限 + 交叉感染风险 + 长者在地老化需求 → HaH 结构性刚需

关键数据速览： | 指标 | 数据 | 备注 | |:------|:-----|:-----| | 启动年份 | 2016 年 | 台大医院首创 | | 运营模式 | 医院主导 + 多学科团队 | 医师/护理师/药师/呼吸治疗师 | | 核心服务 | 在宅急症照护 / 居家安宁 / 慢性伤口 | 2024 年 NHI 试办后急性服务纳入健保 | | 全国 HaH 收案 (2024.7–2026.4) | 7,788 人次 | 全台 192 团队 / 873 家院所 | | 治疗完成率 | 89.9% | 照护后 14 日内急診率仅 2.9% | | 台大医院院长 | 余忠仁 | 2025.11 公开呼吁扩大在宅医疗 | | 关键学术领军 | 施至远 (金山分院) / 黄胜坚 (前院长) | 社区居家安宁模式开创者 |

总体判断：台大在宅是台湾公卫体系 HaH 的 标杆项目——具有临床创新 + 政策影响 + 学术输出三重价值。对 iHomeCare 而言，其最大借鉴在于「公立大医院如何在单一支付方体系下启动并规模化 HaH」。

治理结构

台大医院在宅医疗中心无独立董事会/法人地位，其治理嵌入台大医院体系：

  • 台大医院院长 → 分院院长 → 跨科室在宅医疗协作团队
  • 学术线：台大医学院 → 家庭医学科/公共卫生学院
  • 政策线：台大团队 ↔ 卫福部健保署（参与在宅急症照护政策制定）
  • 无独立关联交易/违规记录——所有财务经台大医院统一管理

D3 — 战略定位

一句话核心逻辑

台大在宅赌的是「超高龄社会的住院替代范式转移」——当台湾 65 岁以上人口突破 20% 且独居老人逼近百万，传统「病人来医院」的模式不可持续，谁先建立可信的「医院级照护进家门」体系，谁就掌握下一阶段医疗资源配置的话语权。

战略差异化要素

维度 台大在宅定位 差异化特征
学术背书 台大医学院 + 公共卫生学院 台湾最顶级医学学术资源，论文发表于国际期刊
临床能力 三甲医学中心全科室支撑 可处理复杂共病患者的急性居家治疗
政策影响力 直接参与 NHI 试办计划制定 卫福部健保署核心咨询方
社区渗透 金山/北护两大分院下沉社区 从城市到偏乡的全覆盖
品牌信任 「台大」品牌 = 台湾民众最高医疗信任 患者接受 HaH 的门槛大幅降低
成本结构 公立医院非营利运营 无商业回报压力，可专注临床质量

  • 2024 年 NHI 在宅急症照护计划以台大经验为蓝本

  • 关键词：急性住院替代、政策 sandbox、健保给付

  • Phase 3（2025–当前）：规模化 + 亚太领导力

  • 2025 年余忠仁在总统层级呼吁扩大在宅医疗
  • 适应症从 3 种感染症扩展至心衰竭/呼吸衰竭/其他感染

  • 关键词：适应症扩展、区域影响力、长照 3.0 整合

D16 — 对 iHomeCare 战略建议

1. 模式借鉴：大医院 vs 社区的 HaH 路径选择

台大在宅提供两条 HaH 路径：

  • 大医院路径（台大总院）：医学中心资源 + 全科室支撑 → 适合复杂共病患者
  • 社区路径（金山/北护分院）：社区医院下沉 + 本地化 → 适合以社区为单位的 HaH

对 iHomeCare 启示：香港同为单一支付方（医管局 HA）体系，建议优先走「大医院 + 社区卫星」混合路径——从 HA 联网医院启动，以社区健康中心/门诊为下沉节点。

2. 适应症起步策略

台大从「3 种感染症 IV 抗生素」起步 → 逐步扩展至心衰竭/呼吸衰竭 → 洗肾/化疗。

对 iHomeCare：建议以「COPD 急性加重 + 心衰竭利尿 + 蜂窝性组织炎 IV 抗生素」为起步适应症（高 incidence + 明确治疗终点 + 低意外风险）。

3. 支付设计：部分负担的经济杠杆

台大模式中，患者 HaH 部分负担仅为住院的 ~1/3（5% vs 10%），形成强经济诱因。

对 iHomeCare：建议向 HA 提议类似的部分负担分级——HaH 日额收费低于住院，刺激患者选择。

4. 不要忽略「在地终老」的人文价值

台大金山分院的社区安宁经验表明：民众接受 HaH 的关键不在费用，而在「在家治疗的安心感」。iHomeCare 需要从一开始就投资于家属教育 + 可信赖的 24h 响应系统。

5. 技术平台不可忽视

台大在宅的技术平台仍较基础（人工调度+通讯软件）。iHomeCare 若能在启动时即建立专用在宅信息系统（调度 + 远程监测 + 家属沟通 + 数据追踪），将成为差异化优势。

6. 护理人力是最大瓶颈

台湾面临严重的护理人力短缺——HaH 虽释放床位但需要额外社区访视人力。香港同样面临医护人力紧张，建议：

  • 早期引入「护理师 + 健康助理」分层人力模型
  • 评估用非医护人力（如社区健康工作者）承担部分非临床任务
